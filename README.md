# PBO-GUI
Pemrograman Berorientasi Objek Menggunakan Bahasa Java di Netbeans IDE 
PBO-GUI merupakan project aplikasi desktop yang dikembangkan menggunakan bahasa pemrograman Java melalui NetBeans IDE. Project ini dibuat sebagai implementasi dasar Pemrograman Berorientasi Objek atau PBO dalam pengembangan aplikasi dengan Graphical User Interface.

Aplikasi menyediakan halaman login yang terdiri dari input username, password, serta tombol login dan keluar. Data yang dimasukkan pengguna akan diperiksa melalui koneksi ke database MySQL. Sistem menggunakan query terparameterisasi untuk mencocokkan username dan password dengan data pengguna yang tersimpan di dalam database.

Apabila data yang dimasukkan sesuai, aplikasi akan menampilkan notifikasi bahwa proses login berhasil beserta nama pengguna. Sebaliknya, apabila data tidak sesuai, sistem akan memberikan pesan bahwa username atau password salah. Aplikasi juga dilengkapi dengan penanganan kesalahan untuk memberikan informasi ketika terjadi masalah pada koneksi database.

Melalui project ini, beberapa konsep dasar pengembangan aplikasi Java diterapkan, seperti penggunaan class dan object, event handling, pembuatan antarmuka menggunakan Java Swing, pengelolaan input pengguna, koneksi database menggunakan JDBC, serta autentikasi data menggunakan MySQL.
